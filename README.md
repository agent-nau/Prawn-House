Prawn House - Seafood Restaurant Website
A modern, responsive website for Prawn House, a bayside seafood restaurant located in Opol, Misamis Oriental, Philippines.

About
Prawn House is a Filipino seafood restaurant offering fresh catches and local favorites with stunning views of Macajalar Bay. This website showcases the restaurant's menu, customer reviews, and location information.

Features
Hero Section: Welcoming banner with restaurant highlights
Menu Highlights: Dynamic menu items fetched from the database
Customer Reviews: Real testimonials from guests
About Section: Restaurant story and values
Contact Information: Location, hours, and directions
Tech Stack
Frontend: React, TypeScript, Tailwind CSS, Framer Motion
Backend: Node.js, Express
Database: PostgreSQL with Drizzle ORM
Styling: Custom ocean-inspired color palette
Restaurant Information
Location: GHCG+6G8, Butuan–Cagayan de Oro–Iligan Rd, Opol, Misamis Oriental
Hours: Open daily until 10 PM
Rating: 4.4/5 stars
Amenities: Dine-in, Takeout, Spacious parking
Getting Started
The application runs automatically. Visit the deployed URL to see the live website.

API Endpoints
GET /api/menu - Retrieve menu items
GET /api/reviews - Retrieve customer reviews
